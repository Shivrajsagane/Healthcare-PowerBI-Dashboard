# 🏥 Healthcare Analytics Dashboard

## 📊 Overview
This Power BI dashboard analyzes hospital data for 500 patients (2023–2024). It provides meaningful insights into patient trends, hospital performance, and revenue analysis.

### Key Insights:
- Total Patients
- Revenue Analysis
- Readmission Rate
- Department Performance
- Patient Demographics

## 🛠 Tools Used
- Power BI
- DAX
- Excel
